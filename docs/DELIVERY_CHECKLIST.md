# Checklist de livraison

## Avant livraison technique

- code fusionné sur `main`
- README relu
- documentation opérationnelle à jour
- variables d'environnement recensées
- tests et build passés

## Avant mise en ligne

- backend joignable
- frontend joignable
- healthcheck OK
- authentification OK
- contact OK
- back-office OK

## Éléments à transmettre à la cliente ou au repreneur

- URL du site
- URL d'administration si distincte
- accès hébergement
- accès base de données / propriétaire Atlas
- accès Cloudinary
- accès email transactionnel
- accès domaine quand il sera acheté
- accès outil de monitoring quand il sera activé

## Éléments à transmettre au prochain développeur

- procédure d'installation locale
- workflow Git
- variables d'environnement nécessaires
- procédure de déploiement
- points en backlog : paiement en ligne, durcissement monitoring, amélioration UX
